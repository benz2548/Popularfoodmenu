เมนูเปาะเปี๊ยะส้มตำ

เมนูอาหารอีสานรสแซ่บ เป็นเมนูสไลต์ใหม่

ส่วนผสม

  • แผ่นเปาะเปี๊ยะญวน 5 แผ่น 

  • ชุดทำส้มตำไทย Lasy Set ตราแม่ประนอม 2 เซต

  • แครอทขูดเส้น 50 กรัม

  • ถั่วฝักยาวหั่นท่อน 50 กรัม

  • กุ้งแห้ง 30 กรัม

  • ถั่วลิสง 25 กรัม

  • น้ำเปล่า (สำหรับแช่เส้นมะละกอ) 2 ถ้วยตวง

  • น้ำเปล่า (สำหรับแช่แผ่นเปาะเปี๊ยะญวน) 2 ถ้วยตวง

    ขั้นตอนการทำ

  • ขั้นตอนที่ 1

    • นำเส้นมะละกออบแห้งใน ชุดทำส้มตำไทย Lazy Set ตราแม่ประนอม มาแช่น้ำเย็น ประมาณ 25 นาที แล้วสะเด็ดน้ำให้แห้ง

  • ขั้นตอนที่ 2

    • นำพริกอบแห้งและกระเทียมอบแห้งใน ชุดทำส้มตำไทย Lazy Set ตราแม่ประนอม มาตำ และตามด้วยนำถั่วลิสงครึ่งหนึ่งมาบุบพอแตก ตักใส่ถ้วยไว้

  • ขั้นตอนที่ 3

    • ใส่มะเขือเทศเชอร์รี กุ้งแห้ง และน้ำปรุงรสส้มตำไทยในชุดทำส้มตำไทย Lazy Set ตราแม่ประนอม ใส่ลงไปในถ้วยที่มีพริกอบแห้ง กระเทียมอบแห้ง ถั่วลิสง และถั่วฝักยาวหั่นแฉลบ คลุกเคล้าให้เข้ากัน

  • ขั้นตอนที่ 4

    • นำแผ่นเปาะเปี๊ยะญวนมาแช่น้ำประมาณ 1 นาที แล้วนำมาวางบนภาชนะ แล้วนำถั่วฝักยาวหั่นท่อน แครอทขูดเส้น และเส้นมะละกอที่สะเด็ดน้ำแล้วนำมาวางตามลำดับ ม้วนโดยพับจากด้านข้างทั้งสองด้านเข้าหาตรงกลาง แล้วม้วนจากด้านล่างขึ้นด้านบน ตัดเป็นชิ้นพอดีคำ เตรียมไว้

  • ขั้นตอนที่ 5

    • จัดเสิร์ฟเปาะเปี๊ยะส้มตำที่หั่นแล้วใส่ในจานคู่กับถ้วยน้ำปรุงรสส้มตำไทย เท่านี้เป็นอันเสร็จเรียบร้อย
